# Imersão Full Stack & FullCycle - Sistema de Venda de Ingressos

## Descrição
Repositório da API feita com Nest.js (Reserva de Ingressos)

## Instalação

```bash
$ npm install
```

## Rodar a aplicação

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
